# EPZ Auto Detailing (Vanilla Clone)

This is a simple 1-page vanilla HTML/CSS/JS site modeled after the Wix page you linked.

## Quick edits

1) **Calendly link**
- Open `script.js`
- Set:
  - `CONFIG.calendlyUrl` to your Calendly booking link

2) **Phone number**
- Open `script.js`
- Update:
  - `CONFIG.phoneDisplay` (how it shows on the page)
  - `CONFIG.phoneTel` (tel: link, include country code)

3) **Hero background image**
- Open `styles.css`
- Replace:
  - `--hero-image: url("...")`

## Run locally
Just open `index.html` in a browser.

If you want a tiny local server (recommended):
- VS Code: right click `index.html` -> "Open with Live Server"
